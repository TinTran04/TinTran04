**Main features:**

- STI test booking, result tracking, and status management
- Menstrual cycle tracking with ovulation and fertile-window prediction

**Tech focus:** React, Tailwind CSS, Spring Boot, REST API, JWT, role-based UI, booking workflow, dashboard UI.
